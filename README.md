# evaluationKotlinD2

Application mobile développée en Kotlin + JetPacK Compose

Sur une première page, elle affiche une liste d'agents du jeu vidéo Valorant. Lorsqu'on clique sur l'un des agents, le téléphone vibre et envoie un son : le son d'une ace dans Valorant.

La seconde page résume la liste des maps dans le jeu vidéo. Elle est présente pour faire fonctionner la navigation, avec la navigation bar.

Pour voir les informations du jeu, l'application utilise l'API Valorant-API.

Il y a aussi un mode thème sombre.
